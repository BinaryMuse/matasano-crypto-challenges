Solutions to Part 2 of the Matasano Crypto Challenges.

Tests
-----

To run the tests:

    mix test

Exercises
---------

To run a specific exercise:

    mix matasano.run <n>

where `<n>` is the exercise number, e.g.

    mix matasano.run 9

To run all the exercises at once:

    mix matasano.run all
